# UHC Spieltagsrechner 🏑

Feldhockey-Turnier-Rechner für den UHC Hamburg.

- Flexible Team-Anzahl (3–8 Teams)
- Jeder gegen Jeden · Live-Tabelle
- Direkter Vergleich als erster Tiebreaker
- Spielverlauf wird automatisch gespeichert
- Ergebnisse exportieren (in Zwischenablage kopieren)
- Dark Mode (automatisch & manuell)
- Druckoptimiert
- LocalStorage-Persistenz

## Nutzung

Öffne `index.html` im Browser oder besuche die GitHub Pages Seite.

## Entwicklung

Einfache HTML/CSS/JS-App ohne Build-Tooling.
